API Notes

This app pulls data from The Movie Database (TMDB) using simple REST endpoints. Popular movies, genre browsing, search, and individual movie details all come from TMDB. Endpoints are used directly in the front end for speed—no extra backend layers.

Used endpoints:

  • /movie/popular
  • /discover/movie?with_genres=GENRE_ID
  • /search/movie
  • /movie/{id}
